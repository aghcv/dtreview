\section{Data and code availability}

The reproducibility repository contains the PROSPERO protocol draft, inconsistency-resolution protocol, current Word manuscript source, SAGE/Overleaf manuscript, extraction workbook, source figure templates, executable analysis code, classification rules, generated LaTeX tables, figures, row-level classifications, and automated quality-control report. The extraction workbook remains the source of truth; derived outputs can be regenerated with \texttt{make analysis}. Public release status, repository DOI, and any restrictions required for included full-text materials should be confirmed before submission.
